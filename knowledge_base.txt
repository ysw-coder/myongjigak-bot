3인 세팅 (성인 2+아동1)

=== 2. 취소 건 정산 관리 ===
월별 정산 관리 (본관)
- 파트너센터 리스트 > 취소 메뉴 > 명지각1956 선택
- 해당 월 취소 내역 결제금액 0원 전수 확인
- 옵션 테스트 건(1,000원 등)은 0원으로 수정 후 결제취소로 변경 저장
- 수수료 발생 건은 0원 처리 금지! 산출내역/고객안내 여부 관리자 메모 기입
- 동관: 모든 내역 결제금액 0원 재확인
- 본관: 롯데 에비뉴엘 환불 처리 여부 확인
- 에어비앤비: 1박당 요금 조정 금액 = 실제 결제 금액 일치 확인
- 완료 후 슬랙에 이모지로 완료 보고

타 OTA 예약 취소 처리 방식
- 스폴 예약: 취소 → 결제금액 환불
- 타 OTA (에어비앤비·온다·놀 등): 결제금액 0원 변경 → 결제취소로 변경 → 예약 취소

- 계좌이체 시 토스플레이스에서 매출 생성 및 현금영수증 발행 필수

지역상품권(남원사랑상품권) 결제 방법
1.  **모바일 QR 결제:**
    *   고객이 남원사랑상품권 앱을 통해 QR 코드를 스캔합니다.
    *   앱에서 명지각(본관)으로 금액을 이체하고, 이체 완료 화면을 직원에게 인증합니다.
    *   직원은 POS 내 **[남원QR]** 버튼으로 결제 처리합니다. (주의: 현금결제가 아닌 [남원QR]로 처리 필수)

2.  **지류 상품권 결제:**
    *   현금과 동일하게 사용합니다.
    *   현금영수증 발급이 가능합니다. (POS 내 현금영수증 발행 처리)
    *   분할 결제 시, 지류 상품권 금액을 현금으로 먼저 처리하고 남은 금액을 카드로 결제할 수 있습니다.
    *   잔돈은 교환되지 않습니다.

3.  **남원사랑카드 결제:**
    *   일반 카드와 동일하게 카드 리더기에 삽입하여 결제 처리합니다.

- 명일 오전조 + 당일 오후조 인수인계 내용 정리 (조식, 정비순서, 특이사항, 이슈 등)
- 얼음 여분 확인 및 만들기 (동절기 상황에 따라 패스)
- 공실 냉난방 확인
- 백오피스, 화장실 라디에이터, 음악 OFF
- 티하우스 조도, 냉난방 조절
- 웰컴티 마감

=== 15. 법인카드 사용 구분 및 지출결의 ===
법인카드 사용 구분
- 상품매입: 매입 상태 그대로 판매하는 상품 (예: 카페 디저트, 주류)
- 소모품_일반: 일반적 운영 비품·소모품 (예: 정비용 휴지, 소모품)
- 소모품_식자재: 조리를 위해 필요한 식자재 (예: 쑥가루, 식재료)
- 주의: 식자재 + 소모품 혼합 결제 절대 금지 — 예산 항목별 구분 결제 필수

지출결의 자금 집행
- 매주 수요일 자정까지 완료된 지출결의 → 매주 금요일 이체
- 긴급 건은 예외 처리 가능

=== 16. 시설 이상 대응 - 냉난방·보일러·온수 ===
1. 어느 거점·객실인지 파악
2. 간단한 재설정 시도 (전원 off→on, 온도 재설정)
3. 해결 안 될 경우 팀장에게 즉시 보고
4. 팀장이 외부 업체 연락
5. 게스트 영향 있을 경우 즉시 안내·사과 문자 발송
6. 노션 로그북에 시설관리 카테고리로 기록

=== 17. 시설 이상 대응 - 누수·소음·진동 민원 ===
1. 현장 확인 후 상태 사진 촬영
2. 팀장에게 즉시 보고
3. 안전 위협 시 해당 객실 이용 중단
4. 게스트 대체 객실 안내 or 환불 협의
5. 노션 로그북 기록

=== 18. 동관 공간별 주의사항 ===
- 동관 리셉션 보일러는 온수용 (난방용 아님)
- 보일러 끄면 티하우스·야외화장실 온수 차단 → 절대 끄지 말 것
- 돌바닥 특성상 온도가 늦게 올라옴 → 입실 전 충분한 사전 가동 필수

=== 19. 조식 운영 ===
시간
- 08:00 조식 세팅 시작
- 08:30~10:30 조식 서빙
- 10:30 조식 마감

식수 인원 공유 절차
- 오픈조: 동관 조식 예약시간 업무일지에 기입
- 마감조: 19:00에 전체 식수인원 정리 → 조리매니저 업무폰으로 문자 발송
- 문자 발송 시 본관 객실 식수인원 포함, 오후 출근 직원 제외
- 주의: '19:00까지'가 아닌 '19:00에' 발송하는 업무 (시간 제한 아님)

=== 20. 체크인·체크아웃 운영 ===
운영 시간
- 기본 체크인: 16:00~17:00 (피크), 검수 완료 객실부터 순차 가능
- 얼리체크인: 객실 정비 후 검수 완료 시 가능 안내
- 체크아웃: 11:00

체크인 안내지 작성 원칙
- 전날 미리 작성 금지 → 당일 오전 A조가 입실 객실 정확히 작성
- 밤 사이 변동사항 발생 가능 → 당일 작성 원칙
- 작성 시 예약 금액 이상 없는지 더블체크 (월말 정산 영향)

=== 21. 예약 확정 체크리스트 ===
- 방막기 설정 확인 (오버부킹 방지)
- 런 옵션 선택 여부 더블체크
- 에어비앤비 입력 시 인원수 반드시 기입
- 온다 예약 확정 시 관리자메모 기입
- 주의: 방막기 미설정으로 오버부킹 발생 사례 있음 → 예약 확정 전 필수 확인

=== 22. 리뷰 관리 ===

- 매일 동관 근무자: 스테이폴리오, 에어비앤비 동관 리뷰 답변 필수

기본 원칙

   - 전용 사용처리 키인코드: 03376227 (명지각)
3. 사용처리 다음날, 롯데 에비뉴엘 파트너센터 전산에 사용내역 정상 조회 확인 (고객 ID 동일 체크)
4. 확인 완료 후 스테이폴리오 파트너센터 예약 객실료 환불처리
5. 고객에게 환불 완료 및 카드사 영업일 기준 최대 7일 소요 안내 문자 발송
